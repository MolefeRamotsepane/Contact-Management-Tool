# Pristine Contact Manager

## Description
A console-based contact manager designed for a small business for managing contacts. This application
is meant for basic information storing proccesses

## Features
- Add contacts, view contacts, delete contacts, update contacts, seacrh contacts
- Continues usage until exit
- Input validation
- Error handling as well
- Contacts are saved into a a json file in order to create persistancy
- File Handling
- CRUD operations

## How to run
- Python 3 should be installed
- Run python app.py
- Follow the on-screen instructions
